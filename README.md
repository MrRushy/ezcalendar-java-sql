## EzCalendar (Team Project)
A Java + SQL calendar management application designed to simplify personal scheduling and group event coordination. The system allows users to manage personal events, create and join groups, and receive group events on their calendars through a shared scheduling workflow.

### My Contribution (Rushil Shanmugam)
For this four-person group project, I contributed to:
- Admin-side functionality for deleting users
- Removing events from user calendars
- Managing/deleting groups within the system
- User interface work for the core program and admin features
- Presentation materials, diagrams, and the final report

### Features
- User sign-up and login
- Personal calendar management
- Group creation and group joining
- Group event creation and event distribution to members
- Admin controls for managing users, groups, and calendar data

### Tech Stack
- Java
- SQL
- IntelliJ IDEA

### How to Run
1. Clone the repository
2. Open the project in IntelliJ
3. Set up the required SQL database/scripts if included
4. Update any database credentials in the project if needed
5. Run the application from the main entry class
